Exercise 1

Create a search filter for a reasonably sized set of words. The full, reasonably sized list of words must be displayed if the user has not inserted any filtering text. A filter text box must allow the user to insert text. The word list must be immediately updated, showing only the words that contain the filter text.

Patterns suggested: observer, mvp, module

Exercise 1

Using Canvas, create a UI that will allow the user to insert shapes such as squares and circles into the canvas and move them around when clicking on them again. The user must have at least two choices for figures. When the user clicks on the canvas area, the selected action must be performed. Selected actions include: insert selected figure into the clicked location; if the user clicked on where a figure is already located, they must be able to drag it and change its position

Patterns suggested: observer, factory, constructor, prototype, interface

Using the div editor from a previous exercise, enhance it with a history stack. Each action the user takes should be stored and be able to be undone; Create a mechanism for the user to define a macro*.
